# wifi-comm
A simple ESP8266-based device for sending text messages using wireless mesh network.
## Key mapping
|     |     |     |      |
|-----|-----|-----|------|
|1    |abc2 |def3 |Power on / off|
|ghi4 |jkl5 |mno6 |Backspace|
|pgrs7|tuv8 |wxyz9|Toggle upstream|
|.?!  |(space)0|N/A| Send / confirm|
